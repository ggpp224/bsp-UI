概要介绍

企业管理员负责管理本企业员工的学习，具备系统管理员为其配置的相应权限，企业商学院能够为管理员提供的核心功能示意图如下：

登录平台

  1. 在浏览器中输入登陆网址：http://xxx.lms.ambw.net；

    备注：

    每个企业客户都具有独立的二级域名，此二级域名在系统管理员新增客户的过程中进行设置，即企业管理可以根据自身需要，定制专属自身企业的域名。

  2. 键入用户名和密码，点击按钮，进入平台。

管理用户数据

支持通过"部门"、"岗位"、"用户分组"3种方式灵活对人员进行管理，即一个用户具有"所属部门"、"所属岗位"和"所属分组"3种分类属性，其中"所属部门"是必有项，其他两个是可选项。 备注："新增用户"操作只能在"用户管理>按部门"下进行，以保证新增个人用户入口的唯一性，在完成用户新增操作后，可以为该用户指定"所属岗位"和"所属分组"。

如何建立和维护部门 / 机构分类体系？

点击"用户管理>按部门"二级菜单，在页面左侧是部门 / 机构分类树的呈现区域：

  1. 导入部门目录树：对应图标为，在初始化情况下，支持用户通过导入目录树的方式建立目录树，能够导入目录树的前提是各分类目录节点及其子节点下没有用户；
  2. 新增目录节点：选中某一节点，然后点击图标，则在该节点下新增节点，新增的节点隶属于所选中的节点；
  3. 移动目录节点：选中某一节点后，点击图标，进行"上移"、"下移"，改变该节点在所属父节点下的前后显示顺序；
  4. 编辑目录节点：选中某一节点，点击图标，编辑该节点的名称；
  5. 删除目录节点：选中某一节点，点击图标，删除该节点，节点可删除的条件是，该节点及其子节点下没有用户。

如何建立和维护岗位分类体系？

点击"用户管理>按岗位"二级菜单，在页面左侧是岗位分类树的呈现区域：

  1. 导入岗位目录树：对应图标为，在初始化情况下，支持用户通过导入目录树的方式建立目录树，能够导入目录树的前提是各分类目录节点及其子节点下没有用户；
  2. 新增目录节点：选中某一节点，然后点击图标，则在该节点下新增节点，新增的节点隶属于所选中的节点；
  3. 移动目录节点：选中某一节点后，点击图标，进行"上移"、"下移"，改变该节点在所属父节点下的前后显示顺序；
  4. 编辑目录节点：选中某一节点，点击图标，编辑该节点的名称；
  5. 删除目录节点：选中某一节点，点击图标，删除该节点，节点可删除的条件是，该节点及其子节点下没有用户。

如何建立和维护分组？

点击"用户管理>按分组"二级菜单，在页面左侧是分组列表的呈现区域：

  1. 导入分组列表：对应图标为，在初始化情况下，支持用户导入分组列表，能够导入分组列表的前提是各分组下没有用户；
  2. 新增分组：点击图标，进行新增分组操作；
  3. 移动分组：选中某一节点后，点击图标，进行"上移"、"下移"，改变该节点在分组列表中的前后显示顺序；
  4. 编辑分组：选中某一分组，点击图标，编辑该分组的名称；
  5. 删除分组：选中某一分组，点击图标，删除该分组，分组可删除的条件是，该分组下没有用户。

维护和管理用户

  1. 新增用户

    进入"用户管理>按部门"二级菜单，点击按钮，进行新增用户操作：按页面要求填写用户信息：

    按页面要求填写用户信息：

    • 基本信息：按要求填写相关字段，"用户姓名"、"登录ID"和"有效期限"是必填项；
    • 组织信息：设置该用户的归属部门、岗位及分组，其中"所属部门"是必选项，其余两项是选填项；
    • 雇佣信息：按要求填写相关字段，此处信息都是选填项；
    • 联系方式：按要求填写相关字段，"E-mail"是必填项。

    上述信息填写完整后，点击"保存"按钮，完成新增用户操作。

  2. 编辑用户

    已经生成的用户，可通过如下方法进行编辑：

    • 点击用户姓名，进入到用户信息详情页面，点击页面下方的按钮。
    • 点击"用户管理"一级菜单下的"按部门"、"按岗位"或"按分组"二级菜单，在页面的列表显示区域，点击该用户"操作"列的图标，进行编辑用户操作。
  3. 调整用户所属部门、岗位和分组
    • 调整所属部门

      点击"用户管理>按部门"二级菜单：

      在用户列表显示区，勾选需要调整所属部门的用户，可以选择多个用户，然后点击按钮，弹出部门调整弹出窗口：

      该弹出窗口左侧显示已经选中的用户；

      在右侧部门目录树中通过鼠标单击，确定要设置的归属部门节点；

      点击"保存"按钮，完成所选用户的所属部门调整操作。

    • 调整所属岗位

      点击"用户管理>按岗位"二级菜单：

      在用户列表显示区，勾选需要调整所属岗位的用户，可以选择多个用户，然后点击按钮，弹出岗位调整弹出窗口：

      该弹出窗口左侧显示已经选中的用户；

      在"岗位分类树"勾选要设置的归属岗位节点，可以选择多个岗位，已选岗位将出现在右侧"已选岗位"处，可以点击每个选中岗位后面的，取消选择该岗位。

      点击"保存"按钮，完成所选用户的所属岗位调整操作。

    • 调整所属分组

      点击"用户管理>按分组"二级菜单：

      在用户列表显示区，勾选需要调整所属分组的用户，可以选择多个用户，然后点击按钮，弹出分组调整弹出窗口：

      该弹出窗口左侧显示已经选中的用户；

      在"分组列表"勾选要设置的归属分组，可以选择多个分组，已选分组将出现在右侧"已选分组"处，可以点击每个选中分组后面的，取消选择该分组。

      点击"保存"按钮，完成所选用户的分组调整操作。

      备注：

      如果需要批量调整用户的所属部门、岗位或分组，点击"按部门"、"按岗位"或"按分组"二级菜单下的、和按钮，分别弹出对应的窗口，通过excel导入的方式批量为用户设置所属部门 / 岗位 / 分组。

      以"批量调整部门"为例：

      点击"下载模板"文字链接，下载批量调整部门的excel模板。

      弹出窗口中为您提供导出相应原始数据的快捷入口：

      导出用户部门数据：导出平台现有用户归属部门的全部数据；

      导出用户数据：导出平台所有用户数据；

      导出部门数据：导出平台部门分类体系数据。

      通过导出以上几种基准数据，可以方便地帮助您摘取原始数据，将相关数据填入所下载的导入模板中，填写完成后，将该excel表格导入平台，即完成批量为用户调整所属部门的操作。

  4. 为用户重置密码

    点击"用户管理"一级菜单下的"按部门"、"按岗位"或"按分组"二级菜单，勾选需要重置密码的用户，点击按钮，弹出密码重置弹出窗口，在弹出窗口中设置所选用户的新密码。

  5. 控制用户的"启用"和"停用"状态

    对于在有效期内的用户，如有需要管理员可以通过点击用户列表区域该用户"操作"列的图标（停用用户）和图标（启用用户），控制该用户的停用和启用状态，当用户被停用后，将不能登录平台。

  6. 批量导入用户

    进入"用户管理>按部门"二级菜单，操作按钮区如下所示：

    点击"批量导入用户"按钮，弹出批量导入用户弹出窗口：

    点击"下载模板"文字链接，下载导入用户的原始模板。

    弹出窗口中为您提供下载平台用户数据和部门数据的快捷链接，供填写导入模板时查询相关信息。

    将模板填写完成后，将文档导入平台，点击"导入"按钮，完成批量导入用户操作。

  7. 查询用户

    进入到"按部门"/"按岗位"/"按分组"二级菜单后，在页面的查询搜索区，键入搜索条件，点击"查询"按钮，将在列表显示区呈现查询结果。

  8. 导出用户数据

    进入到"用户管理"一级菜单后，点击链接，可将当前系统的用户数据导出。导出格式为excel文档。

管理和维护"在线课件"类学习资源

进入到"资源管理>内容管理"二级菜单后，鼠标移动到按钮后，将出现一下拉项，点击其中的"在线课件"，进入到新增在线课件操作页面：

第一步：输入课件基本信息；

第二步：上传课件包，文件必须满足以下格式要求：

  1. 上传SCORM1.2标准的课件压缩包，文件大小控制在200M以内；
  2. PIF文件中必须至少包含SCORM标准所规定的课程描述文件：imsmanifest.xml，并且该文件必须在根目录下；
  3. 若imsmanifest.xml含有中文字符，请保证该xml文件的encoding属性值为UTF8；
  4. ZIP文件中不要包含文件名为中文的文件。

第三步：按字段要求补充课件信息：设置该课件的归属目录节点，填写和补充该课件的其他信息。点击"保存"按钮，完成新增在线课件操作。

备注：

新上载的在线课件不会立即生效，需等待系统管理员审核通过后，才能正式上线使用。

对于系统管理员尚未审核的在线课件，管理员可以重新上载课件包，将原来的课件包替换，通过如下方式操作：在资源列表中，该在线课件对应的"操作"列点击"重新上载课件包"文字链接，弹出"重新上载课件包"操作窗口：

如需要编辑在线课件的基本信息，可通过如下方式操作：

  1. 在资源显示列表中，点击该在线课件名称的文字链接，进入到在线课件详情页面，点击页面底端的按钮，进入到在线课件信息编辑页面；
  2. 在资源显示列表中，点击该课件"操作"列的图标，进入到在线课件信息编辑页面。

如果需要删除课件，则在资源列表中，该课件"操作"列点击"删除"文字链接，将该课件删除，课件可删除的条件是，该在线课件没有通过培训实施操作分配给个人用户。

管理和维护"同步课程"类学习资源

进入到"资源管理>内容管理"二级菜单后，鼠标移动到按钮后，将出现一下拉项，点击其中的"同步课程"，进入到新增同步课程操作页面：

按照页面要求，填写完整各项信息，点击"保存"按钮，完成新增同步课程操作。

已经生成的同步课程，可以点击资源列表中，该试卷"操作"列的图标，编辑该同步课程；

管理和维护"试卷"类学习资源

进入到"资源管理>内容管理"二级菜单后，鼠标移动到按钮后，将出现一下拉项，点击其中的"试卷"，进入到新增试卷操作页面：

第一步：输入试卷基本信息，按照页面要求逐个字段填写；

第二步：录入试题，将该试卷的全部题目录入平台，可通过一下两种方式来录入试题

页面操作：

点击"单选题"、"多选题"或"判断题"按钮，来增加对应类型的试题。

点击"分隔符"，灵活加入要在试卷中呈现的文字。

excel导入：下载平台提供的导入模板，按照模板要求，填写完整后导入平台，完成批量向平台导入试题操作。

录入试题操作完成后，点击页面下方的"保存"按钮，完成新增试卷操作。

已经录入平台的试卷，可在资源列表呈现区域，该试卷对应的"操作"列的图标，编辑该试卷。

如果该试卷尚未被"培训实施-发起在线测试"调用，则可以进行删除操作。

如何调整学习资源的所属分类

进入"资源管理>内容管理"二级菜单，勾选需要调整分类的学习资源，点击按钮，弹出调整资源所属分类的操作框，在弹出窗口中选择要将资源调整到的目录节点，点击"确定"，完成调整资源所属分类操作。

如需要批量调整资源的所属分类，即同时调整多个资源的所属分类，点击按钮，弹出批量调整资源所属分类的操作框：

点击"下载模板"文字链接，下载导入模板；

可以点击"导出平台现有资源分类数据"文字链接，将平台现有全部资源分类数据导出，了解系统当前资源归属分类情况。

在下载的导入模板中将数据填写完整后，将文档导入平台，完成批量调整资源所属分类操作。

管理和维护"文档资料"类学习资源

进入"资源管理>文档管理"二级菜单：

点击按钮，进入到新增文档资料操作界面：

按要求填写相关字段；

平台支持的文档格式如下：pdf、doc、docx、ppt、pps、pptx、ppsx、pot、potx、ls、xlsx、zip、rar；

所上传的单个文档大小要控制在5M以内。

填写完整文档信息，完成文档上载操作后，点击页面下方的"保存"按钮，完成此次新增文档操作。

已经上传的文档资料可以进行"下载"和"重新上载"操作，重新上载后，原有文档将被替换。

管理员所上传的文档，将出现在个人用户端的"课程中心>文档资料"二级菜单中。

管理和维护"讲师"信息

进入"资源管理>讲师管理"二级菜单：

点击按钮，进入到新增讲师操作界面：

按照系统要求，将相关字段填写完整，点击页面底端的"保存"按钮，完成新增讲师操作。

已经生成的讲师信息，如需进行编辑，则在讲师列表中该讲师对应的"操作"列点击，编辑该讲师的基本信息。

备注：

讲师信息用于"培训实施-发起在线同步课程"和"培训实施-发起面授课程"中选择"外部讲师"时进行数据调用。

需求调查和问卷管理

此处为管理员提供面向学员发起问卷调查，为收集学员的培训需求和培训反馈建议提供工具。

"问卷调查"和"培训反馈问卷库"的基本功能

问卷调查：用于管理员面向个人学员灵活地发起问卷调查操作；

培训反馈问卷库：集中管理供各种"培训实施"设置培训反馈的问卷。

发起问卷调查的基本操作流程

  1. 管理和维护"问卷"

    点击按钮，进入到新增问卷操作，按照页面要求，填写信息，选择"页面操作"或者"excel批量导入题目"录入问卷：

    页面操作：点击页面的"单选题"、"多选题"和"简答题"按钮，分别通过页面操作逐道录入题目；

    excel批量导入题目：下载平台提供的模板，在模板中录入题目，将录入完整的excel导入到平台。已经添加的问卷将出现在"问卷调查"二级菜单的显示列表中。

    复制问卷：勾选某个已经添加完成的问卷后，可以点击，将选中问卷复制。

  2. 发起问卷调查

    新添加的，尚未发布的问卷，可以点击问卷列表中，该问卷"操作"列对应的"发布调查"，将问卷发布给特定对象，进入到发布调查页面后：

    设定调查时效、作答方式、公开方式等基本信息；

    设定调查对象：点击按钮，弹出设置培训对象的窗口，在窗口中选择要参与该项调查的部门 / 岗位 / 分组 / 个人用户，勾选要添加的部门 / 岗位 / 分组 / 个人，点击按钮，将选中的部门 / 岗位 / 分组 / 个人用户列入到调查范围。即所设定的调查对象可以是特定部门 / 岗位 / 分组 / 个人，也可以是它们的组合。

    上述操作完成后，点击按钮，完成发布调查操作。

    发布调查操作完成后，被列入到调查对象中的个人用户，将在其首页右侧区域的"调查问卷"模块，查看到该条问卷调查邀请，点击进入参与调查。

  3. 查看调查结果

    问卷调查进行中和结束后，可以点击该问卷"操作"列对应的"查看调查结果"，查看本次问卷调查的结果反馈数据。支持查看"整体统计"和"按人查看"：

    整体统计：查看该次问卷调查的整体统计数据，参与度以及统计结果分析和汇总；

    按人查看：支持逐个查看每个完成调查的人的反馈数据。

    进行中的调查可以进行"关闭调查"操作，关闭调查后，尚未参加调查的培训对象将看不到此问卷调查邀请。

制订和管理"培训计划"

如何制订培训计划？

进入到"培训计划"一级菜单后，点击按钮，进入到制订培训计划页面：

  1. 按照页面要求填写相关字段，如计划名称、目标、描述文字等；
  2. 设置培训对象：按钮，弹出设置培训对象的窗口，在窗口中选择该培训计划要作用的部门 / 岗位 / 分组 / 个人用户，勾选要添加的部门 / 岗位 / 分组 / 个人，点击按钮，将选中的部门 / 岗位 / 分组 / 个人用户列入到培训对象范围。即所设定的调查对象可以是特定部门 / 岗位 / 分组 / 个人，也可以是它们几种的组合；
  3. 设定必修资源和选修资源：点击按钮，分别弹出设置必修资源和选修资源的窗口，在窗口中，勾选该培训计划的必修 / 选修资源，点击按钮，将该资源列入到必修/选修资源列表中；

    备注：

    选修资源中只可以选择"在线课件"类学习资源。

  4. 设定该培训计划的预计起止时间。

完成上述操作后，点击按钮，完成制订培训计划操作。

如何实施培训计划？

培训计划制订后，可以到培训计划显示列表中，点击该培训计划名称前面的，查看该培训计划中所包含的全部资源，点击培训计划名称的文字链接，进入培训计划详情页面，查看该培训计划的详细信息。

如您所制订的培训计划中，只包含在线课件类学习资源，或者只包含1个试卷 / 同步课程资源，则您可以直接点击该培训计划名称后面的图标，进行培训实施操作。

如您所制订的培训计划中，是在线课件、试卷和同步课程类学习资源的组合，或者包含2个以上的试卷/同步课程资源，直接点击该培训计划名称后面的图标，可以将在线课件类的培训资源直接实施生效，其余各学习资源需要点击该培训计划前面的，点击对应资源后面的图标，来单独进行实施操作。

进行"确认实施"操作后，将对终端个人学员生效。

查看培训计划统计报表

状态为"已实施"（所包含的全部资源都已完成培训实施）和"部分实施"（所包含的部分资源完成培训实施）的培训计划，可以点击该培训计划所对应的"操作"列的图标，查看该培训计划的执行统计数据，方便您掌握该培训计划的实际实施情况。

进行"培训实施"--为个人用户分配学习任务

"培训实施"一级菜单是管理员集中管理为学员分配的各种学习任务的统一入口。

"培训计划"、"培训实施"概要介绍及二者之间的关系

"培训计划"是指管理员通过平台进行管理的培训计划：即预计在特定期间，预计向选定的培训对象（学员）配置相应的学习资源。

"培训实施"是指管理员将特定学习活动指定给特定培训对象（学员）的操作：

备注：

培训计划本身不会对个人学员端产生作用，培训计划实施后，才会对个人学员生效。

即培训计划是协助管理员制订和管理计划的工具，真正培训活动实施的操作通过"培训实施"来完成。

平台所支持的培训实施类别简介

平台支持的培训实施类别包含如下5种：

  1. 分配在线课件：指派学员进行在线课件学习；
  2. 发起在线测试：指派学员参加在线测试；
  3. 发起线下测试：指派学员在指定时间到指定地点参加考试；
  4. 发起在线同步课程：指派学员在指定时间登陆平台进行实时在线同步学习，即通过视频会议的形式进行远程学习；
  5. 发起面授课程：指派学员在指定的时间到指定地点参加面授课程学习。

备注：

平台目前能够支持的类型是以上5种，您所在企业的具体权限根据您提出的需求由系统管理员进行设定，即您可能拥有其中部分类型的权限。

各种类型培训实施与资源类型的对应关系

图示中标识了各种不同类型的培训实施与不同类型学习资源的对应关系。

如何分配在线课件？

通过分配在线课件操作，将特定在线课件指定给特定学员。

进入到"培训实施"一级菜单，鼠标移动到按钮上，将出现下拉项，在下拉项中选择"分配在线课件"，进入到分配在线课件操作页面。

进入到分配在线课件页面后，首先填写基本信息，如实施名称、目标、描述文字等；

可以选择该培训实施所关联的培训计划，备选的培训计划列表来自"培训计划"菜单。

  1. "页面操作"方式和"excel批量配课"操作方式介绍

    平台为管理员提两种方式进行课件分配操作：

    页面操作：支持通过页面操作，所见即所得的形式为选定培训对象分配在线课件；

    excel批量配课：支持通过excel导入的方式，为特定部门 / 岗位 / 分组 / 个人用户分配在线课件，适用于用户数据量相对较大的情况。

    备注：

    在一次配课操作中，只能选择其中的一种方式进行。

  2. 通过页面操作的方式设置培训对象和必修/选修课件

    在页面中选择"页面操作，点击按钮，页面将分配课件操作区域展开，在此区域中设置该次课件分配的培训对象和必修 / 选修课件。

    • 设置培训对象：即此次课件分配给哪些用户群体或者个人

      点击"培训对象"处的按钮，弹出设置培训对象弹出窗口：

      此窗口左侧有"选择部门"、"选择岗位"、"选择分组"和"选择个人"4个tab标签：勾选左侧的部门节点，点击按钮，将该部门列入到此次课件分配的作用对象范围。

      管理员可以分别选择该次课件分配的作用部门 / 岗位 / 分组 / 个人，也可以选择培训对象是这几种情况的组合。

      点击"保存"按钮，完成设置培训对象操作。

    • 设置必修和选修课件：

      说明：管理员所设置的必修课件将直接出现在学员端"我的学习"菜单下的"在线课件"标签中；所设置的选修课件将出现在学员端"课程中心>课件资源"二级菜单下，学员进行"选课"操作后，该课程将出现在"我的学习"菜单下的"在线课件"标签中。

      点击"必修课件"处的按钮，设置必修课件：

      在左侧勾选要选择的课件，点击按钮，将该课件列入到必修课件列表，已选课件将出现在弹出窗口的"已选课件"处，可以点击，单个剔除某个已选课件。

      点击"选修课件"处的按钮，设置选修课件，此处操作与设置必修课件相同。

      备注：

      上述操作完成后，点击"分配课件"操作区域的按钮，将此次操作中的配课信息保存，这些信息将呈现在如下截图区域：

      备注：

      管理员在一次分配在线课件实施操作中，可以多次点击按钮，每次操作完成后的数据都将返回到页面中进行显示，按照操作的先后顺序，从上到下依次显示，可以点击每一行操作记录数据后面的"编辑"和"删除"按钮进行相应的编辑和变更操作。

  3. 通过"excel批量配课"方式分配在线课件

    管理员可以选择"excel批量配课"方式分配在线课件，操作界面如下：

    提供"按部门批量配课"、"按岗位批量配课"、"按分组批量配课"和"按个人批量配课"4种方式。

    备注：

    一次操作过程中只可以选择其中一种操作方式。

    以按部门批量配课为例：点击"部门批量配课"tab标签，点击"下载导入模板"后的文字链接下载导入模板，点击"导出在线课件列表"后的文字链接将所在企业拥有权限的在线课件全部导出，点击"导出部门数据"后的文字链接下载平台现有部门分类数据。管理员从"在线课件列表"和"部门数据"表格中查询原始数据，将下载模板填写完整后，导入平台，完成此次配课操作。

    "按岗位批量配课"、"按分组批量配课"和"按个人批量配课"与"按部门批量配课"的基本操作相同。

  4. 设定该次在线课件学习的有效期限

    培训对象和必修 / 选修课件操作完成后，设置"起止日期"，即此次操作在设定的起止时间内有效。即终端个人用户在设定的有效期内可以进行在线课件的学习和选课操作。

如何发起在线测试？

管理员通过"发起在线测试"操作，面向特定学员发起在线测试。

进入到"培训实施"一级菜单，鼠标移动到按钮上，将出现下拉项，在下拉项中选择"发起在线测试"，进入到发起在线测试操作页面：

企业管理员为学员分配的在线测试将出现在个人学员端"我的学习"菜单下的"在线测试"tab标签中。时间到管理员所设定的开始时间之后，个人用户可以点击进行在线测试。

  1. 基本字段含义说明
    • 选择试卷：在此处选择该在线测试所调用的试卷资源，即在"资源管理"菜单下存储的试卷资源；
    • 实施名称：输入此次发起在线测试操作的名称，如"岗位能力测试"，方便进行后续查询和检索；
    • 关联计划：选择此次培训实施所关联的培训计划，备选培训计划的范围为含有尚未确认实施的试卷类学习资源的培训计划；
    • 目标：输入此次在线测试要实现的预期目标；
    • 描述：输入此次在线测试的描述信息；
    • 测试时长：输入本次在线测试的时间长度，即为个人用户设定的答题时间限制，时长单位为分钟；
    • 测试时效：设定本次在线测试的有效期，即个人用户可以参与在线测试的时间段；
    • 测试模式：

      整卷：呈现给个人用户的是一整张卷子的形式，题目从上到下在一个页面上呈现；

      单题：呈现给个人用户的是弹出窗口的形式，每个弹出窗口中只出现一道题目。

    • 允许参加次数：此次在线测试允许个人用户参与的次数，即可以进行几次测试；
    • 分数设置：

      使用试题原始分数，不折算总分：即采用所调用的试卷中各个试题的原始分数，不折算分数；

      按原试题的分数比例，将总分折算为XX分：按照所调用的试卷中各个试题的原始分数比例，将该试卷的满分折算为您所要求的分数，通常情况下，设置满分为100分。

  2. 设置测试对象

    设置测试对象，及设置要参加该项在线测试的用户。

    点击"测试对象"处的按钮，弹出设置测试对象弹出窗口：

    此窗口左侧有"选择部门"、"选择岗位"、"选择分组"和"选择个人"4个tab标签：勾选左侧的部门节点，点击按钮，将该部门列入到此次在线测试的作用对象范围。

    管理员可以分别选择该次在线测试的作用部门 / 岗位 / 分组 / 个人，也可以选择测试对象是这几种情况的组合。

    点击"保存"按钮，完成设置测试对象操作。

  3. 设置通过条件

    设置此次在线测试的达标分数和优秀分数；

    此处设置的达标分数和优秀分数将在该在线测试的统计报表中体现，统计达到标准和优秀的人数。

  4. 设置测试效果反馈调查

    如果管理员需要在此次在线测试完成后，面向测试对象发起问卷调查，则在培训反馈处进行设置，

    "选择问卷"的下拉列表中调用的是"需求管理>培训反馈问卷库"二级菜单下的问卷；

    设定在培训完成后X天内，要求参加测试的人员完成该问卷调查。

    如设定了培训反馈问卷调查，则在所设定的该次在线测试的截止时间之后，可以到"培训实施"菜单下，鼠标移动到该条培训实施记录上，将在该条记录对应的操作列将出现一个浮出层，里面显示各种可以进行的操作，点击"查看反馈结果"文字链接，可查看该次问卷调查的结果。

  5. 查看在线测试统计报表

    有学员参加了改在线测试之后，该在线测试的统计数据将出现在"统计报表>培训实施报表"中的"统计数据"tab标签下的显示列表中，可以点击查看其详细统计数据。

如何发起线下测试？

管理员通过"发起线下测试"操作，面向特定学员发出线下测试通知，测试结束后将测试成绩导入平台，进行测试结果记录。

进入到"培训实施"一级菜单，鼠标移动到按钮上，将出现下拉项，在下拉项中选择"发起线下测试"，进入到发起线下测试操作页面：

企业管理员为学员分配的在线测试将出现在个人学员端"我的学习"菜单下的"线下测试"tab标签中。

  1. 基本字段含义说明
    • 实施名称：输入此次发起线下测试操作的名称，如"职业技能考试"；
    • 关联计划：选择该次在线测试所关联的培训计划，备选培训计划为所有含有尚未确认实施的学习资源的培训计划；
    • 目标：输入进行此次线下测试的目标；
    • 描述：输入此次线下测试的描述文字；
    • 测试时间：设置此次线下测试的起止时间；
    • 地点：输入此次线下测试的测试地点。
  2. 设置测试对象

    设置测试对象，即设置要参加该项在线测试的用户。

    点击"测试对象"处的按钮，弹出设置测试对象弹出窗口：

    此窗口左侧有"选择部门"、"选择岗位"、"选择分组"和"选择个人"4个tab标签：勾选左侧的部门节点，点击按钮，将该部门列入到此次在线测试的作用对象范围。

    管理员可以分别选择该次在线测试的作用部门 / 岗位 / 分组 / 个人，也可以选择测试对象是这几种情况的组合。

    点击"保存"按钮，完成设置测试对象操作。

  3. 上传试卷、导入成绩

    线下测试类的培训实施，允许管理员将考试用的试卷上传到平台，以备日后查验。

    进入"培训实施"菜单，鼠标移动到该条培训实施记录上，将在该条记录对应的操作列将出现一个浮出层，里面显示各种可以进行的操作，点击"上传试卷"/"导入成绩"文字链接，分别进行上传时间和导入测试成绩操作。

  4. 设置测试效果反馈调查

    如果管理员需要在此次在线测试完成后，面向测试对象发起问卷调查，则在培训反馈处进行设置，

    "选择问卷"的下拉列表中调用的是"需求管理>培训反馈问卷库"二级菜单下的问卷；

    设定在培训完成后X天内，要求参加测试的人员完成该问卷调查。

    如设定了培训反馈问卷调查，则在所设定的该次在线测试的截止时间之后，可以到"培训实施"菜单下，鼠标移动到该条培训实施记录上，将在该条记录对应的操作列将出现一个浮出层，里面显示各种可以进行的操作，点击"查看反馈结果"文字链接，可查看该次问卷调查的结果。

  5. 查看线下测试统计报表

    如果管理员已经完成了"导入成绩"操作，则该线下测试的统计数据将出现在"统计报表>培训实施报表"中的"统计数据"tab标签下的显示列表中，可以点击查看其详细统计数据。

如何发起在线同步课程（在线实时同步学习）？

在线同步课程即通过视频会议形式实现的所设定的培训对象的在线实时同步学习。

进入到"培训实施"一级菜单，鼠标移动到按钮上，将出现下拉项，在下拉项中选择"发起在线同步课程"，进入到发起在线同步课程操作页面：

企业管理员为学员分配的在线测试将出现在个人学员端"我的学习"菜单下的"在线同步课程"tab标签中。在管理员所设定的起止时间之内，个人用户可以点击"进入课程"，通过视频会议进行实时同步学习。

备注：

管理员所发起的在线同步课程不会立即生效，需系统管理员审核通过后，才能生效。

  1. 第一步 输入基本信息
    • 选择课程：选择本次实时在线学习的课程，调用的存储在"资源管理>内容管理"二级菜单下的同步课程类
    • 学习资源，同步课程本质上存储的课程的基本信息；
    • 实施名称：输入此次在线同步课程的实施名称，方便进行后续查询和检索；
    • 关联计划：选择该培训实施所关联的培训计划；
    • 目标：输入此次在线实时同步学习的目标；
    • 描述：输入此次在线实时同步学习的描述信息；
    • 起止时间：设定此次课程的起止时间；
    • 授课教师：设定此次课程的授课教师；
    • 内部讲师：从"用户管理"中企业自有用户中进行选择；
    • 外部讲师：从"资源管理> 讲师管理"列表中选择本次课程的讲师。
  2. 第二步 设置培训对象

    点击"培训对象"处的按钮，弹出设置培训对象弹出窗口：

    此窗口左侧有"选择部门"、"选择岗位"、"选择分组"和"选择个人"4个tab标签：勾选左侧的部门节点，点击按钮，将该部门列入到此次在线同步课程的作用对象范围。管理员可以分别选择该次在线同步课程的作用部门 / 岗位 / 分组 / 个人，也可以选择培训对象是这几种情况的组合。

    点击"保存"按钮，完成设置培训对象操作。

  3. 第三步 上传培训资料、设置培训反馈
    • 如果管理员需要在上课之前上传一些此课程的相关资料，可在第三步操作中完成，单个文档大小控制在5M之内，支持格式如下：

      pdf

      Word：doc/docx

      Powerpoint：ppt/pps/pptx/ppsx/pot/potx

      Excel：xls/xlsx

      压缩包：zip/rar

    • 设置培训反馈，即在课程结束后面向参与学员进行效果调查

      "选择问卷"的下拉列表中调用的是"需求管理>培训反馈问卷库"二级菜单下的问卷；

      设定在培训完成后X天内，要求参加课程的人员完成该问卷调查。

      如设定了培训反馈问卷调查，则在所设定的该次在线同步课程的截止时间之后，可以到"培训实施"菜单下，鼠标移动到该条培训实施记录上，将在该条记录对应的操作列将出现一个浮出层，里面显示各种可以进行的操作，点击"查看反馈结果"文字链接，可查看该次问卷调查的结果。

  4. 查看在线同步课程统计报表

    如果平台中有此次在线同步课程的学员学习数据，则该次在线同步课程的统计数据将出现在"统计报表>培训实施报表"中的"统计数据"tab标签下的显示列表中，可以点击查看其详细统计数据。

管理和维护社区话题

  1. 编辑话题

    管理员可以编辑所有人发布的话题，点击该话题对应的图标编辑该话题。

  2. 移动话题

    管理员可以将特定话题移动到其他版块，点击该话题对应的图标移动该话题。

    管理员可以进行删除话题操作，点击该话题对应的图标，执行删除话题操作。

  4. 置顶/取消置顶

    管理员可以将特定话题进行"置顶"，让其出现在和页面最顶端，点击该话题对应的图标，进行置顶操作，如需要取消该话题的制定，则点击该话题对应的图标。

控制学员的发言权限

进入到某个话题的详情页面之后，页面左侧是该话题的发表和回复人的详细信息，在用户姓名的下方有"禁言"操作文字链接，管理员可以设置禁止该人发言。

所有被禁止发言的人员将汇总在"社区管理"右上角的中，管理员可以在此查看和恢复禁言人员的发言权限。

统计报表

查看培训分析报告

培训分析报告对整体培训数据进行汇总和分析。

以月为统计单位，以图表作为表现方式，统计"在线课件参与人数"、"在线课件学习时长"、"人均在线课件学习时长"、"人均在线课件学习次数"、"人均在线课件学习门数"、"培训实施活动次数统计"以及"各类培训实施参与人数"。

分成"概览"、"在线课件"、"试卷"和"同步课程"4个查询标签：

"在线课件"、"试卷"和"同步课程"标签分别统计3种不同类型资源的应用和实际学习 / 测试数据。

此报表分成"按部门查看"、"按岗位查看"、"按分组查看"和"按个人查看"，与用户的不同属性相对应，管理员可以按其中的任意一种方式查询配课数据。

查看培训实施报表

对各种类型培训实施的分布状态进行统计。

查看学习数据报表

"学习数据报表"将各种不同类型的学习活动进行汇总统计。

"概览"标签对个汇总不同各类型的学习活动的基本数据进行概要统计，支持"按部门查看"、"按岗位查看"、"按分组查看"和"按个人查看"4种查询方式；

"在线课件"、"在线测试"、"线下测试"、"在线同步课程"和"面授课程"5个标签分别统计这5种不同类型学习活动的学习数据，同样支持"按部门查看"、"按岗位查看"、"按分组查看"和"按个人查看"4种查询方式。

系统管理

学员角色管理

所在企业的默认管理员，即系统管理员在为您创建账号的时候所设置的管理员，可以将公司其他成员设置为企业管理员，辅助其进行学习和培训管理。

进入到"系统管理>角色管理"二级菜单后，点击按钮，弹出设置窗口，选择要设置为企业管理员的用户，点击弹出窗口中的，完成新增管理员操作。

如果需要撤销某个管理员的权限，可以到管理员列表中的"操作"列点击"解除授权"图标。

学员端"积分"和"排名"功能启用/停用状态设置

进入"系统管理>功能设置"二级菜单：

此处控制积分和排名的停用和启用状态，即管理员可以控制学员端是否能使用积分功能、是否能看到排名数据。

启用积分功能后，学员可到"个人中心-我的积分"中查看积分值及积分明细；

启用排名功能后，学员可到"个人中心-我的排名"中查看个人排名和组织排名数据，个人学习和组织排名信息将对学员完全公开。

积分规则管理

进入"系统管理>积分规则管理"二级菜单：

管理员在此处控制每一种学员学习 / 互动行为是否计入积分，并可以变更每一项学员学习 / 互动行为的积分值。

学习排名管理

进入"系统管理>学习排名管理"二级菜单：

点击页面左侧的标签，切换查看"个人排名"和"组织排名"：

个人排名：查看所有个人用户的学习排名数据；

组织排名：切换查看"部门排行榜"、"岗位排行榜"和"分组排行榜"，分别查看各个部门 / 岗位 / 分组的学习排名数据。

问题反馈管理

进入"系统管理>问题反馈管理"二级菜单：

此处汇总个人用户向管理员发出的咨询 / 反馈，集中答复个人用户的咨询和疑问。

发布和管理新闻公告

进入"系统管理>新闻公告"二级菜单：

此处用于管理员面向企业内所有用户发布新闻公告。

查看管理日志

进入"系统管理>管理日志"二级菜单：

此处支持管理员按照功能模块查看每一个模块相关的操作记录。

定制企业LOGO

进入"系统管理>定制LOGO"二级菜单：

企业管理员可以上传自身企业的LOGO，所上传的LOGO将出现在该企业下所有用户的左上角。
